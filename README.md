# Bucketlist Planner

Bucketlist Planner is a travel-themed React application where users can plan, organize and document backpacking trips.

## Features

- Add and delete trips
- Upload up to 4 images
- Slideshow gallery
- Track trip status
- Store trips using LocalStorage
- Travel-inspired UI design

## Technologies

- React
- JavaScript
- CSS
- Vite
- LocalStorage

## Run Locally

```bash
npm install
npm run dev
```
